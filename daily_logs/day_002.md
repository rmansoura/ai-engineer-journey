### 2. NumPy Basics
In your notebook, run the following cells:

```python
import numpy as np
import pandas as pd

# 1. NumPy Arrays vs Lists
# Arrays are faster and allow element-wise math
python_list = [1, 2, 3]
numpy_array = np.array([1, 2, 3])

print(python_list * 2)   # Repeats the list: [1, 2, 3, 1, 2, 3]
print(numpy_array * 2)   # Math operation: [2, 4, 6]
```

### 3. Pandas DataFrames
DataFrames are like Excel sheets in code.

```python
# 2. Creating a DataFrame
data = {
    'Name': ['Alice', 'Bob', 'Charlie', 'David'],
    'Age': [24, 27, 22, 32],
    'City': ['New York', 'Los Angeles', 'Chicago', 'Houston'],
    'Salary': [70000, 80000, 55000, 90000]
}

df = pd.DataFrame(data)
df  # Displays the table
```

### 4. Inspecting Data
```python
# 3. Quick Inspection
print("Shape:", df.shape)  # (Rows, Columns)
print("\nInfo:")
print(df.info())           # Data types and missing values
print("\nStatistics:")
print(df.describe())       # Summary stats for numerical columns
```

### 5. Selection and Filtering
```python
# 4. Selecting Data
print(df['Name'])          # Select a single column
print(df[['Name', 'Age']]) # Select multiple columns

# 5. Filtering Rows
# Who is older than 25?
older_than_25 = df[df['Age'] > 25]
print("\nPeople older than 25:")
print(older_than_25)
```
