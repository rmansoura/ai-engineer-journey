# AI Engineer Journey

This repository documents the journey of becoming an AI Engineer. 

## About
- **User**: Former TSE returning to the workforce.
- **Goal**: AI Engineer.
- **Mentor**: Antigravity AI Agent.

## Structure
- `ROADMAP.md`: The detailed plan.
- `daily_logs/`: Directory to store daily work and notes.
- `projects/`: Directory for code projects.

## Getting Started
Check out [ROADMAP.md](ROADMAP.md) to see the plan.

## Quick Start
1.  **Install dependencies**:
    ```bash
    uv sync
    ```
2.  **Launch Jupyter Notebook**:
    ```bash
    uv run jupyter notebook
    ```
    Or simply open a `.ipynb` file in VSCodium.
